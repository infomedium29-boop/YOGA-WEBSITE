YOGA PREMIUM LIGHT WEB STRANICA

Sadržaj ZIP-a:
- Višestranični statički web: index, o meni, programi, raspored, galerija, kontakt, politika privatnosti, politika kolačića
- Premium light dizajn s responsive navigacijom i hamburger izbornikom
- Scroll animacije kroz IntersectionObserver
- Cookie banner
- SEO meta opisi, robots.txt i sitemap.xml
- Svi generirani vizuali su u AVIF formatu i nalaze se u assets/images

Što treba zamijeniti prije objave klijentu:
1. VAŠ LOGO u headeru/footeru zamijeniti stvarnim logom ili nazivom studija.
2. Kontakt podatke info@vasstudio.hr, +385 00 000 000 i adresu zamijeniti stvarnim podacima.
3. U sitemap.xml i robots.txt zamijeniti https://www.vasa-domena.hr stvarnom domenom.
4. Kontakt forma trenutno koristi mailto:info@vasstudio.hr. Za profesionalno slanje bez otvaranja e-mail aplikacije spojiti Web3Forms/Formspree/Cloudflare Worker ili drugi backend.
5. Tekst politike privatnosti i kolačića je demo i treba ga uskladiti sa stvarnim podacima poslovnog subjekta.

Deploy:
- Upload cijelog sadržaja foldera na GitHub repository.
- Povezati repository s Cloudflare Pages.
- Build command ostaviti prazno.
- Output directory ostaviti kao root folder projekta.
